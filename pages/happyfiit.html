Happy Fiit

Fitness app

build with
React and ExerciseDB API

Concept

The concept was to create a simplified fitness app that offers a range of exercises for each muscle group in the body, and promotes overall fitness. I aimed to create a platform that tracks your physical and mental well-being, enabling you to be in control of your health.

I wanted to build a user-friendly application adept at offering tailored exercises based on user input, integrating features that simplify the fitness journey. I believe fitness doesn't have to involve lengthy workouts, rather, it's the consistent day-to-day routines that change us for the better.

Wireframe

I started the project by creating a wireframe of the app, which helped me a lot to visualize the website's layout, define functionalities, and significantly expedite the development process.

Technologies

The technologies I used for this website include React, Vite, NodeJS, Git & GitHub, Excalidraw, ExerciseDB API from Rapid API for fitness-related data, Bootstrap and Netlify.
